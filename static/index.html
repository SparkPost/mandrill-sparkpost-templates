Mandrill Template Translation

This is a Mandrill to SparkPost template translation service. From here you can translate a template from file or migrate one directly from your Mandrill account into SparkPost.

MIGRATE A TEMPLATE » TRANSLATE TEMPLATE TEXT »

Supported Features

  • Basic Handlebars syntax:
    variables, logic, iteration
  • Logic and iteration:
    if, each
  • Mandrillisms:
    elseif, backtick conditionals `x < 10`
  • Inline helpers:
    upper, lower and title are accepted but ignored
  • MailChimp merge tags:
    *|variable|*

Unsupported Features

  • HTML escaping:
    please put your markup in your template ;)
  • Inline helpers:
    url, date, striptags
  • Block helpers:
    unless, with
